Aufgabe 5: Schachbrett mit HTML und JavaScript erzeugen Dauer: 20 Minuten Ziel: Erstellen eines einfachen Schachbretts mit HTML und JavaScript, um die DOM-Manipulation zu üben. Erstelle eine Datei aufgabe5.html und füge das grundlegende HTML-Grundgerüst ein. Erstelle ein div mit der ID chessboard im HTML-Bereich, in dem das Schachbrett dargestellt wird. Füge ein script-Tag hinzu und schreibe JavaScript-Code, der ein 8x8 Schachbrett erzeugt. Nutze Schleifen, um die 64 Felder des Schachbretts zu erstellen und abwechselnd schwarze und weiße Quadrate (z. B. als div-Elemente) einzufügen. Verwende CSS im style-Tag oder in einem separaten CSS-File, um jedes Quadrat gleich groß zu machen und farblich abzuwechseln.
