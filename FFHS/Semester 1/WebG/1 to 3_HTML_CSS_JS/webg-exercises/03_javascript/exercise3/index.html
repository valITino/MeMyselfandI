Aufgabe 3: FizzBuzz mit erweiterten Regeln Dauer: 15 Minuten Ziel: Üben von Schleifen und Bedingungen in JavaScript. Erstelle eine Datei aufgabe3.js. Schreibe eine Funktion fizzBuzz, die alle Zahlen von 1 bis 50 durchläuft. Für jede Zahl gelten folgende Regeln: Ist die Zahl durch 3 teilbar, soll „Fizz“ ausgegeben werden. Ist die Zahl durch 5 teilbar, soll „Buzz“ ausgegeben werden. Ist die Zahl durch 3 und 5 teilbar, soll „FizzBuzz“ ausgegeben werden. Ist die Zahl durch 7 teilbar, soll „Bazz“ ausgegeben werden. Für alle anderen Zahlen wird einfach die Zahl ausgegeben. Teste die Funktion, indem du fizzBuzz() aufrufst und das Ergebnis in der Konsole ausgibst.
